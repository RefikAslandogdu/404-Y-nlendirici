# 404 Yonlendirici

404 URL'lerinizi aktif sayfalariniza akilli sekilde eslestiren web araci.

## Calistirma

**Windows:** `baslat.bat` dosyasina cift tiklayin.

**Mac:** `baslat.command` dosyasina cift tiklayin.

Gerekli paketler otomatik yuklenir ve tarayici otomatik acilir. Baska birsey yapmaniza gerek yok.

**Mac kullanicilari icin not:** Ilk calistirmada macOS guvenlik uyarisi verebilir. Terminali acip su komutu calistirin:

```bash
xattr -d com.apple.quarantine baslat.command
```

Ardindan dosyaya tekrar cift tiklayin.

## Nasil Calisir

1. Sol panele aktif sayfalarinizin (200 OK) URL'lerini yapisitirin
2. Sag panele yonlendirmek istediginiz 404 URL'leri yapisitirin
3. "Analiz Et" butonuna tiklayin
4. Sonuclari inceleyin, isterseniz hedef URL'leri dropdown'dan degistirin
5. "Excel Olarak Indir" ile sonuclari `.xlsx` olarak indirin
